 # 🚢 Titanic Survival Predictor

ML модель которая предсказывает выживет ли пассажир Титаника.

## Технологии
- Python
- Scikit-learn (Random Forest)
- Flask API
- Docker

## Как запустить локально
docker pull davidsedrakyan/titanic-api:v1
docker run -p 5000:5000 davidsedrakyan/titanic-api:v1

Открой браузер: http://localhost:5000
